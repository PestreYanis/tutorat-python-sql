# Tutorat Python & SQL

Progression quotidienne sur 6 semaines : bases Python et SQL jusqu'à un niveau intermédiaire (jointures avancées, sous-requêtes, fonctions de fenêtrage, POO, modélisation, optimisation), avec un projet fil rouge en fin de parcours.

## Projet fil rouge (semaines 5-6)

Application Python en ligne de commande connectée à une base SQLite : CRUD complet, requêtes avancées (jointures, agrégations, fenêtrage), import/export CSV, code organisé en modules.
→ `semaine-05-06/`

## Progression

| Semaine | Contenu |
|---|---|
| 1 | Fondations Python (variables, conditions, boucles, listes) + révision SQL (SELECT, agrégations) |
| 2 | Structures de données Python (dict, tuples) + jointures SQL (INNER, LEFT, self-join) |
| 3 | Fonctions, gestion d'erreurs, fichiers CSV + sous-requêtes SQL (corrélées, EXISTS) |
| 4 | POO, modélisation relationnelle, fonctions de fenêtrage, optimisation (index, EXPLAIN) |
| 5-6 | Projet fil rouge : application Python + SQLite |

## Structure

```
semaine-XX/jour-YY-sujet/
```
Un dossier par séance, code des exercices correspondants.

## Stack

Python 3, SQL (SQLite)
